{%- comment -%} Reusable "hopping agent" widget. A duck emoji hops square-to-square across a 5x5 grid along a fixed path. Dependency-free, scoped CSS, multi-instance safe. Each hop is a parabolic arc (vertical apex above the target square + horizontal displacement) sold with a scaling ground shadow. Path: grid[2][0] -> grid[2][4] (rightward), then grid[2][4] -> grid[0][4] (upward), one square per hop, looping. Usage: {% include widgets/agent-hop.html %} {%- endcomment -%}
